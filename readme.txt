== Installation ==

1. Upload `optimized-product-photos` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to admin panel (/wp-admin/) and you will find Pixc app at WooCommerce -> Pixc
